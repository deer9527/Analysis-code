项目标题：Analysis code for: Failure of early lymphocyte recovery identifies sepsis patients with initial lymphopenia at highest risk for late mortality
主要研究内容：
Background: Sepsis-induced lymphopenia is associated with adverse outcomes, but static single-point assessments fail to capture the prognostic value of dynamic lymphocyte count evolution.
Methods: This retrospective cohort study extracted data from the MIMIC-IV database. Adult septic patients with initial lymphopenia (<1.0×10⁹/L) were included. Group-Based Trajectory Modeling (GBTM) identified lymphocyte count trajectories over the first 7 days. Multivariable Cox and logistic regression were used to assess associations with 7-day, 28-day, ICU, and in-hospital mortality.
Code contents: This repository contains all R code used for data extraction, GBTM modeling, statistical analyses.
